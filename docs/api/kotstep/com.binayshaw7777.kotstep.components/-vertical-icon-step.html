VerticalIconStep

@Composable
fun VerticalIconStep(modifier: Modifier = Modifier, stepTitle: String?, isCurrent: Boolean, isVisited: Boolean, isCompleted: Boolean, lineThickness: Dp = 1.dp, stepSize: Dp = 28.dp, stepIcon: ImageVector, stepShape: Shape, incompleteColor: Color = Color.Gray, completedColor: Color = Color.Blue, checkMarkColor: Color = Color.White, stepTitleOnIncompleteColor: Color = checkMarkColor, stepTitleOnCompleteColor: Color = completedColor, stepIconsColorOnComplete: Color = checkMarkColor, stepIconsColorOnIncomplete: Color = completedColor, showCheckMarkOnDone: Boolean = false)

Create a composable representing a single step in a vertical icon-based stepper with customizable appearance and behavior.

Displays a single step in a vertical icon-based stepper, allowing customization of colors, icons, labels, and icons based on its state (current, visited, or completed).

Parameters

modifier

The modifier for styling the composable. (Optional)

stepTitle

The title or description of the step. (Optional)

isCurrent

Whether the step is currently active or not. (Required)

isVisited

Whether the step has been visited (prior to the current step) or not. (Required)

isCompleted

Whether the step is completed or not. (Required)

lineThickness

The thickness of the connecting line between steps. Defaults to 1.dp. (Optional)

stepSize

The size of the step icon. Defaults to 28.dp. (Optional)

incompleteColor

The color for incomplete steps. Defaults to Color.Gray. (Optional)

completedColor

The color for completed steps. Defaults to Color.Blue. (Optional)

checkMarkColor

The color of the checkmark symbol for completed steps. Defaults to Color.White. (Optional)

stepIcon

The icon to display for the step. (Required)

stepTitleOnIncompleteColor

The color of step titles on incomplete steps. Defaults to checkMarkColor. (Optional)

stepTitleOnCompleteColor

The color of step titles on completed steps. Defaults to completedColor. (Optional)

stepIconsColorOnComplete

The color of step icons on completed steps. Defaults to incompleteColor. (Optional)

stepIconsColorOnIncomplete

The color of step icons on incomplete steps. Defaults to checkMarkColor. (Optional)

showCheckMarkOnDone

Whether to display a checkmark icon for completed steps. (Optional)
